Regression

Regression is basically predicting a target value by using independent predictors. We use regression to find a relationship between our values.

Linear Regression: Shows the linear relationship between independent value and the dependent value(in this example, region sales and global sales).

Random Forest Regression: simply combines multiple decision trees and tries to find the output based on average of these decision trees.

Ridge Regression: Is one of the advanced versions of linear regression, it is better for long term predictions.
